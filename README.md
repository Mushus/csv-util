# csv-util
csvをごにょごにょするコード群

#How to use

#### To add record simply.

```
ruby csv-simple-union.rb header=no input=inputfile[,inputfile2...]
```
* header: The csv has header.
* input: Input files.

#### To add column simply. 

```
ruby csv-simple-join.rb input=inputfile[,inputfile2...]
```

* input: Input files.
